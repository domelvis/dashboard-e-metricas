Sobre o Projeto

Este projeto é uma plataforma robusta para análise e visualização de métricas. Ele utiliza uma arquitetura de microsserviços com **Node.js** para a API, **Python** para o motor de análise e **PostgreSQL** para o banco de dados.

Nosso objetivo é fornecer uma ferramenta escalável e eficiente para ajudar empresas a tomarem decisões baseadas em dados.
